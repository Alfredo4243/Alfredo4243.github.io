¿Qué es HTML?

HTML es el lenguage básico de programación de sitios web, no es muy dificil de utilizar ni de aprender y es necesario para poder utilizar la mayoría de los demás lenguages de programación de sitios web. Básicamente, este lenguage se encarga de crear el texto y algunas funciones de la página web (como por ejemplo el índice). Por desgracia, hay un problema en este tutorial. Al intentar poner ejemplos de etiquetas (se explicara más adelantelo que es), no he podido ya que al escribirlos, el editor de código las toma como una etiqueta y no como parte del texto. Por ello, en los ejemplos que ponga de etiquetas en vez de estar rodeados de estos signos "< >", estaran rodeadas de estos "[ ]". Sabiendo esto, la estructura básica de HTML es la siguiente [etiqueta (atributo)] relleno de la etiqueta [/etiqueta]. El ejemplo de verdad sería el que se muestra en la primera imagen de la derecha. Al principio pondré ejemplos de estas, pero despues, excepto algunas extepciones, no lo haré ya que el formato siempre es el mismo. Por último, para programar una página web con cualquier lenguaje informático es necesario un editor de código, que se pueden descargar gratis o pagando en internet.

Lo básico de HTML

Lo primero que se debe saber para empezar a trabajar con HTML es saber que es una etiqueta. Las etiquetas son las cosas que hacen que el texto salga de una manera o de otra y es lo único que se utiliza en HTML. Hay muchas, tantas que no podré mencionar todas. A veces, las etiquetas contienen atributos, pero esto se verá más adelante. Para empezar, hay que conocer las etiquetas imprescindibles para que HTML funcione. Simplemente, hay que escribir lo que se muestra en la imagen de la derecha. No hay que copiar las etiquetas title y h1, solo hay que escribir eltítulo que quieras dentro de la etiqueta title y sustituit h1 por todo lo demás que se quiera añadir. Hay muchas mas etiquetas, por ejemplo, las que sirven para añadir texto. Hay varias maneras para hacer esto y las más utilizadas son las siguientes:

  1. Para los títulos, lo mejor es utilizar las etiquetas h1 o h2. (ejemplo: [h1]texto[/h1]) Al escribir esto, el texto tendrá tamaño de título . Hay hasta el h6, pero este ya no es para titulares y no es muy utilizado (h1 es el más grande y h6 el más pequeño)
  2. Para escribir párrafos, se utiliza la etiqueta p (ejemplo [p]texto del párrafo[/p]). Con los párrafos se pueden hacer muchas cosas, por ejemplo, esta lista está dentro de un párrafo. Es recomendable utilizar la etiqueta p y no otras ya que, en el caso de no utilizarla, puede causar problemas al trabajar con otros lenguajes.
  3. Si se quiere añadir texto sin ningun otro tipo de interés, se puede usar la etiqueta div. Esta etiqueta es simplemente un contenedor. No obstante, se puede utilizar para otras cosas.
  4. Por último, si se quiere excribir un texto citado hay que escribir la etiqueta cite. Dentro de esta debes poner solamente la cita. Debajo de esta etiqueta, se debe poner otra llamada blockquote y dentro de esta se debe poner el texto que se ha citado en la etiqueta anterior.

Para editar el texto se suele utilizar CSS, pero en HTML tambien hay algunas maneras. Por ejemplon las etiquetas más utilizadas para esto son b(negrita), u (subrrayado), i (cursiva)... Esto no es muy importante ya que se hace mejor con CSS. Si se quiere dar un salto de linea, hay una etiqueta especial para esto. Esta tiene un formato diferente a las demás ya que se cierra sola (ejemplo: [br /]). Por último, es importante saber hacer listas y tablas ya que son muy simples e importantes.

  1. Listas desordenadas: Se debe creal la etiqueta ul, en cuyo interior se deben añadir varias etiquetas li (una etiqueta li por cada elemento de la lista) y dentro de cada etiqueta li se puede añadir que añadir una etiqueta a, dentro de la cual irá el texto. La etiqueta a sirve principalmente para añadir un enlace a cada elemento de la lista. Además, se pueden poner listas dentro de listas dividiendo así los elementos de la primera lista en los de la segunda. Se hace colocando una lista dentro de la etiqueta ul de otra lista y debajo del elemento de esta que se quiera subdividir.
  2. Listas ordenadas: Es igual que las listas ordenadas solo que hay que cambiar la etiqueta ul por la etiqueta ol
  3. tablas: Las tablas son muy parecidas a las listas solo que el nombre de las etiquetas cambia. Primero, hay que creat una etiqueta table, dentro de la cual hay que añadir varias etiquetas tr (cada etiqueta tr es cada columna) y, finalmente, hay que añadir en su interior varias etiquetas td (cada número de elementos de cada columna), dentro de las cuales va el texto.

Conceptos más avanzados de HTML

HTML tiene muchos conceptos avanzados. Uno de los más sencillos es añadir imágenes. Para esto, se suele crear una etiqueta div o una p en cuyo interior contenga una etiqueta img, la cual tiene una estructura especial [img atributo /]. Despues de hacer esto, hay que añadir un atributo a la etiqueta img. Esto se puede hacer de dos maneras. La primera es escribir el atributo "src="enlace de la imagen"" (no hay que poner las comillas exteriores). La segunda es descargarse la imagen, guardarla en una carpeta y escribir el siguiente atributo "src="nombre de la carpeta en la que se encuentra la imagen/nombre de la imagen.formato de la imagen"". Para que esto funcione, es muy importante que la imagen se encuentre dentro de una carpeta la cual esté dentro de la misma carpeta que la página web. A la etiqueta img se la pueden añadir además otros atributos como "title="el texto que se quiera mostrar al pasar el cursor por encima de la imagen"" , o "width="un valor"" (en este último es recomendable poner porcentages y no pixeles ya que los porcentages se adaptan mejor a pantallas de diferente temaño). Si se quiere redirigir a alguien a otra página web, hay que utilizar la etiqueta a. Esta debe contener el atributo href, el cual ya hemos visto (quedaría así: href="enlace"). Tambien se puede añadir el atributo "target="blank"", el cual hace que en vez de redirigirte a otra página web en la misma pestaña lo haga abriendo otra, de tal manera que no cambie la pestaña en la cual está la página web que te redirige. (Es muy importante que, además de atributos, te etiqueta a contenga un pequeño texto, el cual será lo que se lea en la página web (mira el ejemplo de la derecha)) Sabiendo esto, ya se puede aprender a hacer un menú y un índice.

Un índice se crea haciendo una lista ordenada o desordenada la cual debe encontrarse preferiblemente dentro de una etiqueta nav. Dentro de cada etiqueta li debe haber una etiqueta a, en la cual se debe poner el texto que se quiere que se lea. También debe contener un atributo href, el cual debe contener un id. Un id es un atributo que le da nombre de alguna manera a una etiqueta. Si se le pone un id a una etiqueta (se pone escribiendo id=cualquier palabra o letra), cada vez que se escriba ese id, el editor de código sabrá que te refieres a esa etiqueta. Con lo cual, sabiendo que el atributo href se encarga de buscar lo que se le pida, si se le pone un id, el href te debería llevar automáticamente al id que se haya especificado. Ningun id debe ser identico a otro distinto (el id siempre tiene que ir introsducido por #)(quedaría así:[li] [a href="#CH"]texto que se quiere que se lea[/a][/li]).

Con los conocimientos que ya se han explicado se podría crear sin necesidad de que lo explique, pero igualmente lo voy a explicar. Es muy simple, solamente hay que hacer una lista igual que la del índice. La única diferencia es que en el atributo de href hay que poner el enlace en el que hay que poner el enlace a tu otra página web.
